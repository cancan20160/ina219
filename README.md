# INA219 Current/Power Monitor Driver

Taşınabilir, donanım bağımsız INA219 sensör sürücüsü.
INA219, I2C protokolü ile haberleşen şönt direnç üzerinden akım ve güç hesabı yapan Texas Instruments yapımı dijital bir sensördür. Shunt ve Bus gerilimlerini 9 ile 12-bit arasında ölçme ve 128'e kadar örnekleme kabiliyetlerine sahiptir.

README güncellenecek, platform, test ve example dosyaları eklenecektir.

## Özellikler

- Platform bağımsız tasarım
- Tüm INA219 register adreslerine erişim
- Runtime kalibrasyon
- Birden fazla senseör desteği
